Project Overview

This is a production-grade starter project for building secure, multi-tenant SaaS applications using the Next.js App Router. It is intentionally minimal in its dependencies, focusing on the core patterns for security, performance, and maintainability.

- **Authentication**: Secure user login/logout with session management.
- **Multi-Tenancy**: Users belong to organizations, and data is isolated between tenants.
- **Role-Based Access Control (RBAC)**: A centralized policy layer to control user permissions.
- **Workspaces**: Projects and tasks as the core domain model.
- **Collaboration**: Comments on tasks.
- **File Handling**: Secure file uploads and downloads.
- **Auditing**: A log of significant events for security and compliance.
- **Public Sharing**: A mechanism to share resources publicly via a secure token.
